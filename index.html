This map illustrates the average daily particulate matter 2.5 during 2020. Individual readings taken from different counties in each state.

Data was taken from the EPA, sorted for PM2.5 and averaged daily county values for each state. Values were sorted using the Air Quality Index (AQI).

The plots below are visualized using Bokeh, and have been sorted generally acording to region.

Use the toolbox in the upper right corner of the Bokeh plots to zoom out/in or subselect points on the plots to show how they correlate across states.
